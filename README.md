# 台北市嘉禾社區費用管理顯示網頁

這是一個為台北市嘉禾社區設計的管理系統，用於追蹤住戶資料與管理費繳納紀錄。

## 🚀 快速開始

對於全新使用者，請按照以下步驟完成開發環境配置並啟動專案：

### 1. 前置需求 (Prerequisites)

在開始之前，請確保您的電腦已安裝以下軟體：
*   **Node.js**: 建議安裝 [LTS 版本](https://nodejs.org/) (包含 npm)。
*   **Git**: 用於複製專案代碼 (選配)。
*   **VS Code**: 推薦的代碼編輯器。

### 2. 安裝步驟 (Installation)

1.  **取得代碼**：
    將專案下載或複製到您的本地目錄。

2.  **安裝依賴套件**：
    開啟終端機 (Terminal/PowerShell)，進入專案根目錄並執行：
    ```bash
    npm install
    ```

### 3. 啟動專案 (Running the App)

1.  **啟動開發伺服器**：
    執行以下指令啟動本地預覽：
    ```bash
    npm run dev
    ```
2.  **瀏覽網頁**：
    啟動成功後，終端機會顯示伺服器網址（通常為 `http://localhost:3000/`）。請透過瀏覽器開啟該網址。

## 📖 使用指南

### 住戶查詢 (Resident Portal)
*   首頁即為住戶查詢介面。
*   選擇您的 **號碼**、**號碼尾綴 (如 -1, -2, -3)** 及 **樓層**。
*   **動態選單限制**：21、23 號支援 -1~-3 尾綴；13、15、17、19 號僅限「無」及「-1」。
*   點擊「查詢明細」即可查看所有歷史繳費紀錄。

### 管理登記 (Admin Portal)
*   點擊導航欄的 **「管理設定」**。
*   **一般模式**：可選擇住戶並登記當月費用、車位數及繳費期間。系統會自動根據費率計算總額。
*   **主委模式**：點擊右上角「切換主委權限」，可覆蓋/修改已存在的繳費紀錄以處理糾紛。
*   **日期驗證工具 (New)**：
    *   系統會強制確保「起算年月」早於或等於「結算年月」。
    *   **錯誤警告**：若設定不正確，該項目會顯示**紅色驚嘆號警示訊息**，且輸入欄位變為紅色，直到日期修正。
    *   進入新月份時，若舊週期已結束，系統會自動歸位起算年月至當前月份。

## ⚙️ 配置與部署

### API Token 設定
若需使用 AI 相關功能，請複製 `.env.example` 為 `.env` 並填入：
`GEMINI_API_KEY=您的金鑰`

### 自動部署
本專案透過 **GitHub Actions (Native Pages Deployment)** 實現自動化部署。
1.  將變更推送到 `main` 分支。
2.  GitHub Actions 將自動進行 Vite 建置並部署至 **GitHub Pages**。
